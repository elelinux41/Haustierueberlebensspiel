Willkommen beim Haustierüberlebensspiel


Spielen auf Windows

Zuerst musst du dir Python installieren, die Installerdatei lädst du dir herrunter in dem du hier klickst.

Wenn du dann das Haustierüberlebensspiel spielen möchtest, musst du die Datei 'qell' öffnen


Spielen auf Linux

Zuerst musst du dir Python installieren:

Auf debianbasierenden Systemen musst du ins Terminal eingeben: sudo apt install python3.9

Auf RedHat-basierenden Systemen musst du ins Terminal eingeben: sudo dnf install python3.9

Auf Computern mit SUSE musst du ins Terminal eingeben: sudo zypper install python39

Auf archbasierenden Systemen musst du in das Terminal eingeben: sudo pacman -S python

Um das Spiel zu starten musst du dann die Datei 'linx' öffnen.
